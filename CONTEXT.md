# Athena Space Quest

Athena Space Quest is a small Android-wrapped web game whose current purpose is to exercise the Google Play publishing path.

## Language

**Publishing Test**:
A release whose primary goal is proving that the app can pass the Google Play upload and review path, not maximizing player retention or commercial polish.
_Avoid_: Production launch, marketing release

**Play Upload Artifact**:
The signed Android App Bundle intended for upload to Google Play for the current package.
_Avoid_: APK, build output, release file

**Play Package**:
The Google Play Console app record identified by the Android package name `com.inaya.athenaspacequest`.
_Avoid_: App id, namespace, bundle id
